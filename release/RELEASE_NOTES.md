# Connectja Release Notes

## v0.1.3

### Added
- Monaco Editor に現在の `Line / Column / 総行数 / 総文字数` を表示するようにしました。

### Fixed
- Monaco Editor にフォーカスがあるときは、プレビュー側のショートカットよりエディタ側の `Ctrl+C / Ctrl+V` などを優先するようにしました。
- カーソル位置を変えたときに、行・列表示が更新されないことがある問題を修正しました。
- `#SCROLL` など `#` で始まる行は、通常の音符色分けが入らないようにしました。
- 音源再生タイミングと波形の補正値を調整しました。

### Notes
- 既存のサーバー側セキュリティ強化や `v0.1.2` までの修正はそのまま維持しています。
