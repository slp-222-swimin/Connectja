# Connectja 利用ガイド

このガイドは、Connectja のエディタ UI を使って TJA チャートを共同編集する方法を日本語で説明します。

## 1. ルームに参加・作成する

- アプリを開き、表示される入力欄にニックネームを入力します。
- ロビーで新しいルームを作成するか、既存ルームに参加します。

## 2. 音源アップロード

- `Chart Metadata` パネルにある音源アップロード欄で `.ogg` ファイルをアップロードしてください。アプリは Supabase Storage の `audio` バケットに `${roomId}.ogg` というファイル名で保存します。
- アップロード後、プレビューに波形が表示され、再生やシークが可能になります。

## 3. キャンバスの基本操作

- 左クリックで選択中の音符タイプを配置します。
- 右クリック＆ドラッグで複数音符を選択できます。
- 既存の音符をドラッグして移動できます。ロール（5/6/7）を置くと自動的に終端（8）が近接位置に配置されます。
- ヘッダー領域で右クリックするとコマンド追加メニューが出ます（BPM, HS/SCROLL, MEASURE, GOGOSTART, GOGOEND）。
- ヘッダーのシークバーをクリック／ドラッグして再生位置を移動します。
- プレビュー上でマウスホイール（中ボタンの回転）を回すことで水平スクロールできます。

## 4. TJA ソースエディタ

- 右側のパネルに Monaco エディタで生成された TJA ソースが表示されます。
- 必要に応じて直接編集できます。編集後に保存すれば GUI 状態に反映されます。

## 5. コマンドとゴーゴー（Gogo）

- `GOGOSTART` と `GOGOEND` に対応しており、キャンバス上ではオレンジのオーバーレイで視覚化されます。追加はヘッダーのコンテキストメニューから行います。
- BPM 変更、HS（SCROLL）、MEASURE も同様に追加可能です。

## 6. 保存とエクスポート

- `Save (Ctrl+S)` をクリックしてノート／コマンドをデータベースに保存します。
- `Export` ボタンで現在のチャートを `.tja` ファイルとしてダウンロードできます。モーダルのチェックボックスで「エクスポート後に削除」を選ぶと、次の処理が行われます：
  1. Supabase Storage から `${roomId}.ogg` を削除（存在する場合）
  2. 当該ルームの `notes` と `commands` をすべて削除
  3. `rooms` テーブルの `audio_url` を null に更新
  4. ローカル UI 状態をクリア

## 7. ショートカットとヒント

- `Ctrl+S`（または Save ボタン）：TJA ソースと GUI の状態を保存
- ヘッダーの Add Command メニューで正確な位置にコマンドを追加
