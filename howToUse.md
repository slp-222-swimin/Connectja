# Connectja 使い方ガイド

## 1. ルーム
- ロビーでルームを作成/参加します。
- 参加者はリアルタイムで同じ譜面を編集できます。

## 2. 基本編集
- 画面上部のノート種別（1〜7）を選んで配置します。
- 右クリックでコマンド（BPM/HS/MEASURE/GOGO）を追加できます。
- `0` は消しゴムです。

## 3. シークと再生
- ヘッダーをクリック/ドラッグでシークできます。
- シークはグリッドにマグネット吸着します。
  - ドラッグ中: マウス追従
  - 単クリック or ドラッグ終了: 吸着アニメーション
- 再生中は中央固定のプレイヘッド（緑）を表示します。

## 4. グリッドとズーム
- `Grid: 1/n` で分割を変更できます。
- `x0.5 / x1.0 / x2.0 / x4.0` でズーム変更。
- ズーム変更後は現在シーク位置へ自動再センタリングされます。

## 5. 履歴（Undo/Redo）
- `Ctrl+Z` / `Ctrl+Y`（または `Ctrl+Shift+Z`）で Undo/Redo。
- ツールバーの履歴ボタンから履歴一覧を確認できます。
- 履歴項目をクリックするとその時点へジャンプできます。

## 6. 小節ジャンプ
- ステータスバーの `Jump M` に小節番号を入力し `Go`。
- Enter キーでもジャンプ可能です。

## 7. TJA Source
- 右側の Monaco Editor で TJA を直接編集できます。
- `Save (Ctrl+S)` で TJA → GUI 反映。
- `Export` で `.tja` 保存。
- `Import` は `.tja` ファイルアップロードで取り込みます（貼り付け方式ではありません）。

## 8. 音源
- `.ogg` をアップロードすると波形が表示されます。
- 音量は `SEVOL` / `SONGVOL` で調整可能です。

## 9. 再生演出
- ノートヒット時に拡大＋フェード演出を表示。
- 連打/風船の継続区間中は専用アニメーションを表示。
- 高密度時の音割れを抑えるため、SE 側にコンプレッサ/減衰制御を入れています。

## 10. Electron 版
- 開発起動: `npm run electron:dev`
- 配布ビルド: `npm run electron:dist`
- Release には通常 `Connectja Setup *.exe` を使用します。
